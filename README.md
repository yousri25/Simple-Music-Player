# Music Player

This is a simple one-page music player built with HTML, CSS, and JavaScript. It allows you to play, pause, and switch between songs, displays album covers, changes the background with a linear gradient for each song, and lets you search for songs by title or artist. The search results appear in a table, and you can click on a song to play it instantly. Perfect for practicing web development basics.  

The website is live at: https://yousri25.github.io/Simple-Music-Player/
